Detailed Description

Administrative client for Code Compiler & Executor Server.

Author
Rares-Nicholas Popa & Adrian-Petru Enache
Date
April 14, 2025
Version
1.0.0

This application provides administrative interface for managing the Code Compiler & Executor Server. It connects to the admin port (8081) and allows monitoring server status, viewing logs, and shutting down the server.
